Components

A set of VueJS-based components. Use them like this: 
// Import single component to Vue
import { TwoScene } from 'https://rawgit.com/designstem/framework/master/framework.js'
new Vue({
  components: { TwoScene }
})

// Import all components to Vue
import * as components from 'https://rawgit.com/designstem/framework/master/framework.js'
new Vue({
  components: { ...components }
})

Utils

A set of Javascript helper functions. Use them like this:
// Import just some functions
import { random } from 'https://rawgit.com/designstem/framework/master/utils.js'

// Import all under utils namespace
import * as utils from 'https://rawgit.com/designstem/framework/master/utils.js'

// Make all utils available for Vue templates
new Vue({
  methods: { ...utils }
})
